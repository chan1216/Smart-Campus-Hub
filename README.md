# Smart-Campus-Hub 🎓

스마트 캠퍼스 허브 - 대학교 예약 시스템

## 개요

Smart Campus Hub는 대학교 캠퍼스 내 다양한 예약 서비스를 제공하는 웹 애플리케이션입니다.

## 기능

### 1. 상담 예약 시스템 (💬)
- **상담 주제 선택**: 진로, 학업, 개인, 취업 상담 중 선택
- **날짜 선택**: 원하는 상담 날짜 선택
- **시간 선택**: 가능한 시간대 중 원하는 시간 선택
- **예약 확인**: 예약 정보 확인 모달 표시

### 2. 실습실 예약 시스템 (🖥️)
- **실습실/강의실 선택**: 드롭다운에서 원하는 공간 선택
- **날짜 선택**: 예약할 날짜 선택
- **동적 시간표**: 선택한 실습실과 날짜에 따른 실시간 예약 현황 표시
- **다중 시간 선택**: 여러 시간대를 동시에 선택 가능
- **예약 확인**: 예약 정보 확인 모달 표시

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 원하는 예약 시스템 선택 (상담 예약 / 실습실 예약)
3. 안내에 따라 정보 입력 및 선택
4. "예약하기" 버튼 클릭하여 예약 완료

## 파일 구조

```
Smart-Campus-Hub/
├── index.html              # 메인 페이지
├── consultation.html       # 상담 예약 페이지
├── lab-reservation.html    # 실습실 예약 페이지
├── css/
│   └── styles.css         # 공통 스타일
├── js/
│   ├── consultation.js    # 상담 예약 JavaScript
│   └── lab-reservation.js # 실습실 예약 JavaScript
└── README.md              # 프로젝트 설명
```

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
